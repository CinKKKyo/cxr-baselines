Use

    $ ssh -L 8095:localhost:8095 night    # for various values of 8095 

    $ cd /scratch/wboag/2019/cxr/cxr-baselines

    $ . venv_cxr/bin/activate

    $ ipython notebook --no-browser --port=8095
